SuperSearch Settings

VS Code-style find & replace for any web page.

Behavior

Saved

Keyboard Shortcuts

Cmd + F / Ctrl + F — Open find panel (when override is on)
Esc — Close panel
Enter / Shift + Enter — Next / previous match
Alt + C — Toggle Match Case
Alt + W — Toggle Whole Word
Alt + R — Toggle Regex
Alt + L — Toggle Find in Selection (requires selection)
Cmd/Ctrl + Alt + Enter — Replace All

You can also click the SuperSearch toolbar icon to open the panel.
